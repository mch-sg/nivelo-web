# Nivelo

A web-based chat platform built for company-internal communication. Users can create accounts, set up chat rooms, and invite colleagues via a role-based access system.

## Tech Stack

- Backend: PHP
- Database: MySQL
- Frontend: HTML, CSS, JavaScript

## Features

- User authentication (signup, login, password reset)
- Chat room creation with invite-only access
- Session management and user profiles
- Role-based access control per chat room

## Database
Schema available in /db. Covers user management, chat rooms, messages, and session handling.

## Note
This was a high school final project. No longer actively maintained or hosted.
